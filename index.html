Models

It is commonly suggested that a virus spreads in an exponential way until artificial immunization is available (Wikipedia). Hence linear curve fittings problems are solved to test if this hypothesis applies to 2019-nCoV as well.

However, during recent days the original model f(x) performed worse every day, hence an alternative model g(x) is provided that uses a smaller base.

Interpreting the model

This virus won't be the end of civilization, despite the opinions of some conspiracy theorists.

An exponential growth is considered the model for how viruses spread. It sounds scary but it isn't as bad as some suggest. Example: During the Ebola outbreak during 2014, an exponential growth could also be measured. However, after some time the proper reactions kicked in and the number of infections flatlined (Wikipedia).

Hence, my plea: Don't fall for conspiracy theories. Instead, trust the real experts on this topic. As a rule of thumb: If you don't know how to calculate the Root Mean Square Error or don't even know what that is, don't use this model to spread panic.

Background

How This Project Came To Be

I simply was interested in seeing if the Coronavirus spreaded in an exponential way. But as I couldn't find any graphical data, I decided to build my own dataset and publish it on Twitter. However, I assumue my followers aren't as interested in this graphs as I am, hence I decided to rather build this small personal project.

Can You Contribute?

Of course! Feel free to write a Pull Request on Github if you have newer or better data. Simply insert the data in the corresponding table and most importantly include a relieable source.

Sources must be from English or German media organizations.

If you wish to add functionality to this site, feel free as well.
